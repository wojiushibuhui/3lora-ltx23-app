cg down 下载工具使用

安装 cg down

使用 cg down 下载文件需要先安装 codewithgpu 工具包：

pip install codewithgpu

如何使用 cg down 下载？

安装完成后，您可以使用 cg down 命令下载模型文件。

示例：

下载模型文件 umt5-xxl-enc-fp8_e4m3fn.safetensors：

cg down zealman/Wan_2.2/umt5-xxl-enc-fp8_e4m3fn.safetensors

下载目录说明

使用 cg down 下载的文件会默认带有下载链接中的目录文件夹。

例如：

执行命令 cg down zealman/Wan_2.2/umt5-xxl-enc-fp8_e4m3fn.safetensors 后，文件会被下载到当前目录下的 Wan_2.2 文件夹中，完整路径为：

./Wan_2.2/umt5-xxl-enc-fp8_e4m3fn.safetensors

移动到目标目录

如果您希望将文件直接下载到目标目录（而不是在下载链接的目录文件夹中），需要执行以下步骤：

步骤 1： 执行下载命令

cg down zealman/Wan_2.2/umt5-xxl-enc-fp8_e4m3fn.safetensors

步骤 2： 将文件从下载目录移动到上一级文件夹

mv Wan_2.2/umt5-xxl-enc-fp8_e4m3fn.safetensors ./umt5-xxl-enc-fp8_e4m3fn.safetensors

步骤 3： 删除下载时创建的目录文件夹

rmdir Wan_2.2

或者如果目录不为空，使用：

rm -rf Wan_2.2

完整示例：

# 下载文件
cg down zealman/Wan_2.2/umt5-xxl-enc-fp8_e4m3fn.safetensors

# 移动到当前目录
mv Wan_2.2/umt5-xxl-enc-fp8_e4m3fn.safetensors ./umt5-xxl-enc-fp8_e4m3fn.safetensors

# 删除下载目录
rmdir Wan_2.2
